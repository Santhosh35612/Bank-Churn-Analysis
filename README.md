## Overview
This project explores the causes and patterns of customer churn in the banking sector. Using real-world customer data, it analyzes who is likely to leave the bank, why they do so, and how banks can respond.

## Contents
- **Bank Churn Analysis.ipynb**: Jupyter Notebook with all code for data exploration, modeling, and visualization.
- **DS1_C5_S1_BankChurn_Data_Concept.csv**: Raw customer churn dataset.
- **DS1_C5_S1_BankChurn_Data_Dictionary.pdf**: Documentation for dataset columns.
- **Bank-Churn-Analysis-Understanding-Customer-Attrition-Patterns.pptx**: Slide presentation summarizing findings and business recommendations.

## Key Analyses & Insights

- **Exploratory Data Analysis**:  
  - Visualized churn rates by age, account balance, credit score, and geography.
  - Identified high-risk customer segments and trends.

- **Predictive Modeling**:  
  - Built and evaluated models to predict customer churn.
  - Used feature importance to rank the most influential factors in staying or leaving.

- **Business Recommendations**:  
  - Suggested retention strategies for high-value customers.
  - Presented insights with charts and slides for management and marketing teams.

## Getting Started

1. Clone/download the repository and unzip (if needed).
2. Open `Bank Churn Analysis.ipynb` in Jupyter Notebook.
3. Load the provided CSV data and run through the notebook.
4. Review the PPT for executive summaries and recommendations.

## Requirements

- Python 3.x
- Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook
